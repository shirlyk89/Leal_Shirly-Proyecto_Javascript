# Descripcion del proyecto

## FakeStore es una aplicación web de tipo e-commerce desarrollada con HTML, CSS y JavaScript. Consume productos desde la API pública FakeStore API y permite a los usuarios: Visualizar productos dinámicamente en pantalla, Buscar productos por nombre, Filtrar por categorías, Ordenar por precio y nombre, Agregar productos a un carrito de compras, Guardar el carrito en localStorage y Finalizar la compra

## El proyecto aplica manipulación del DOM, uso de módulos JavaScript y persistencia de datos en el navegador.
